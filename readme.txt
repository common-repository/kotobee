Control access to your Kotobee cloud ebooks and libraries using other plugins such as WooCommerce, WooCommerce Subscriptions, and Memberful.

== Description ==

[kotobee website]: https://www.kotobee.com

This is the official plugin from [Kotobee][kotobee website], it allows you to sell access to your [Kotobee][kotobee website] cloud ebooks or libraries right from your store or membership website.

Currently, we support Woocommerce, Woocommerce Subscriptions, and Memberful. If you need us to support other plugins, please don't hesitate to contact us and send us details about your usecase.

== Frequently Asked Questions ==

= Do I need a Kotobee account to be able to use this plugin? =

Yes. Create a free account in [Kotobee website](https://www.kotobee.com)
